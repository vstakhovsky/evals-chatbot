# Stage 03 — Domain-Specific Skills Experiments

**Status:** Planned

## Purpose

Evaluate narrowly scoped skills derived from real Stage 01 and Stage 02 failure modes.

## Initial Candidates

- Retrieval coverage auditor
- Critical-routing reviewer
- Failure-case curator
- Synthetic-query realism reviewer
- Judge-disagreement analyst

## Entry Criteria

- Stable failure taxonomy
- Canonical benchmark results
- Clear input/output contract for every skill
- Benchmark cases assigned to each experiment

## Non-Goals

- No generic "analyze everything" skills
- No skill that merely wraps one Python function
- No agent framework without demonstrated need
